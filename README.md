# Comprehensive Raylib Guide - From Basics to Advanced

Welcome to the complete raylib learning guide! This guide will take you from absolute beginner to advanced raylib developer with fully explained, working examples.

## What is Raylib?

Raylib is a simple and easy-to-use library to enjoy videogames programming. It's designed to be easy to learn, useful for prototyping, and great for beginners. Despite its simplicity, it's powerful enough for professional game development.

## üöÄ Quick Start

### 1. Install Raylib
See `INSTALLATION.md` for detailed instructions for your platform:
- Linux (Ubuntu/Debian/Fedora/Arch)
- macOS (Homebrew/Manual)
- Windows (MinGW/vcpkg)

**Quick install (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install libraylib-dev
```

### 2. Compile and Run Examples
```bash
# Make scripts executable
chmod +x compile.sh compile_all.sh

# Compile a single example
./compile.sh 01_hello_window.c
./01_hello_window

# Test all examples (syntax check)
./compile_all.sh
```

### 3. Start Learning
Begin with `01_hello_window.c` and work through in order. See `INDEX.md` for the complete learning path.
